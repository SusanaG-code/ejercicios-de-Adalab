La importancia de la etiqueta mark

Elemento HTML Mark representa un texto marcado o resaltado como referencia o anotación, debido a su relevancia o importancia en un contexto particular.En una cita () o en otro bloque (

), generalmente indica texto que es de especial interés pero que no está destacado en el material de origen, o material que necesita especial escrutinio, aunque el autor original no lo considerase importante.

Por otro lado, indica una parte del contenido del documento que probablemente sea relevante para el usuario. Por ejemplo, se puede utilizar en una página que muestra los resultados de búsqueda para resaltar cada uno de estos por palabra.

No se debe confundir el elemento mark con el elemento . El elemento se utiliza para denotar intervalos de texto de especial importancia, mientras que el elemento se utiliza para denotar intervalos de texto de especial relevancia.
